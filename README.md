# Beyond the Firmament

> RPG por turnos desarrollado en Unity 3D como proyecto final para la materia Fundamentos de VFX.

---

## Acerca del juego

**Beyond the Firmament** es un RPG por turnos ambientado en un universo de fantasía oscura. El jugador explora entornos únicos, enfrenta enemigos mediante combate estratégico y descubre una narrativa impulsada por misterio cósmico.

El proyecto fue construido con enfoque en efectos visuales en tiempo real, iluminación técnica y optimización de rendimiento dentro del motor Unity.

---

- **Motor:** Unity 6000.3.7f1
- **Lenguaje:** C#
- **Control de versiones:** Git / GitHub Desktop
- **Documentación:** GDD, Manual Técnico, Manual de Usuario

## Cómo ejecutar

1. Clona o descarga este repositorio.
2. Abre **Unity Hub** y selecciona **Agregar proyecto desde disco**.
3. Selecciona la carpeta raíz de este repositorio.
4. Abre el proyecto con **Unity 6000.3.7f1**.
5. En el panel **Project**, navega a `Assets/Scenes/` y abre la escena principal.
6. Presiona **Play** en el Editor de Unity.

# Beyond the Firmament

## About the Game

**Beyond the Firmament** is a turn-based RPG set in a dark fantasy universe. Players explore unique environments, battle enemies using strategic combat mechanics, and uncover a narrative driven by cosmic mystery.

The project was built with a focus on real-time visual effects, technical lighting, and performance optimization within the Unity engine.

## How to Run

1. Clone or download this repository.
2. Open **Unity Hub** and click **Add project from disk**.
3. Select the root folder of this repository.
4. Open the project with **Unity 6000.3.7f1**.
5. In the **Project** panel, navigate to `Assets/Scenes/` and open the main scene.
6. Press **Play** in the Unity Editor.
